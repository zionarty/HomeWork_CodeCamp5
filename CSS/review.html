  1. Simple selectors เลือกจาก ชื่อ ไอดี คลาส
    • Name 
      p {
    color : red ;
}
    • Id 
      #para1 {
    color : red ;
}
    • Class 
      .center {
    color : red ;
}
  2. Combinator selectors เลือกจากตัวที่มีความสัมพันธ์กัน
    • Descendant Selector 
      div p {
    background-color: yellow;
}
    • Child Selector 
      div > p {
    background-color: yellow;
}
    • Adjacent Sibling Selector 
      div + p {
    background-color: yellow;
}
    • General Sibling Selector 
      div ~ p {
    background-color: yellow;
}
  3. Pseudo-class selectors เลือกจาก specified state 
    a:link {
    color : red ;
}
a:hover {
    color : blue ;
}
  4. Pseudo-elements selectors เลือกจาก specified part 
    p::first-line {
    color : red ;
}
  5. Attribute selectors เลือกจากตัวที่มี Attribute 
    a[target] {
    background-color: yellow;
}

    Number Topic Detail
    2.1 property และ value คืออะไรแตกต่างกันอย่างไร
    • Property คือ ชื่อคุณสมบัติ
    • Value คือ ค่าที่กำหนด
    2.2 property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง
    • เปลี่ยนสี 
      h2 {
    color : blue;
}
    • เปลี่ยนขนาด 
      h1 {
    font-size : 30px;
}
    • ตั้งค่า font-family 
      h2 {
    font-family : sans-serif;
}
    • ปรับขนาดรูปภาพ 
      img {
    width : 500px;
}
    • เพิ่มเส้นขอบ 
      table {
    border : 1px solid black; 
}
    • เพิ่มสีพื้นหลัง 
      div {
    background-color : green;
}
    • ปรับช่องว่าง 
      div {
    padding : 5px;
}
    • ปรับระยะขอบ 
      div {
    margin : 5px;
}
    2.3 value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร
    • cm : centimeters
    • mm : millimeters
    • in : inches (1in = 96px = 2.54cm)
    • px : pixels (1px = 1/96th of 1in)
    • pt : points (1pt = 1/72 of 1in)
    • pc : picas (1pc = 12 pt)
